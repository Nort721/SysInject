# SysInject
A utility for testing different kinds of code injection on Windows

![softwaresc](https://github.com/Nort721/ClientWatch/blob/main/docs/images/softwaresc.png)

## Supported Injection Methods
* Basic CRT
* Thread Hijack
* Function Stomp
* Process Herpaderping
* Manual Map
